사용자 수정
1. 금주의 참석인원 
	- 로그인이 안되어있을 시 팝업 후 로그인 confirm 完
2. 팀 구성 
	- 일반사용자 안보이게 完
	- 관리자 아니면 안들어가지게 完
3. 지나간 참석자 명단 
	- 참석자 명단 상세에서 인원 수 보이게 完
	- 팀 보기에서 능력치 제거 完
4. 회비 납부 현황 
	- 멤버만 보이게 完
	- 멤버만 렌더링되도록 비지니스로직 수정 完
5. 자유게시판 
	- 상세보기 ui 변경 完
	- 글 작성 시 미로그인 시 로그인 confirm 完
	- 댓글 작성 시 미로그인 시 로그인 confirm 完
6. 건의게시판 
	- 모두 볼 수 있게 完
	- 글 작성 시 미로그인 시 로그인 confirm 完
	- 멤버만 글 작성 完
